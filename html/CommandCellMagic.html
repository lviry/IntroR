Capturing output¶

You can also capture stdout/err from these subprocesses into Python variables, instead of letting them go directly to stdout/err

In [17]:
%%bash
echo "hi, stdout"
echo "hello,stderr" >&2

hi, stdout

hello,stderr
In [19]:
%%bash --out output --err error
echo "hi, stdout"
echo "hello,stderr" >&2
In [20]:
print(output)
print(error)

hi, stdout

hello,stderr

Background Scripts¶

These scripts can be run in the background, by adding the --bg flag. When you do this, output is discarded unless you use the --out/err flags to store output as above.
